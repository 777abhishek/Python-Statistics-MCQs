📐 Topic 1
NumPy & Statistics Module Q1–Q4
Q1Medium
What is the default value of ddof in np.std()?
💡 Explanationnp.std() uses ddof=0 by default → population standard deviation. Use ddof=1 for sample std with Bessel's correction.
Q2Hard
What is the output of the following code?
import numpy as np marks = np.array([2, 4, 6, 8, 10]) print(np.var(marks))
💡 ExplanationMean=6. Deviations²: 16,4,0,4,16 → sum=40. Population variance (ddof=0) = 40/5 = 8.0.
Q3Medium
Which function gives the 3rd quartile (Q3) for an array?
💡 ExplanationQ3 = 75th percentile = np.percentile(data, 75). Q1=25th, Q2=50th (median), Q3=75th.
Q4Tricky
What does np.corrcoef(x, y)[0][1] return?
💡 Explanationnp.corrcoef returns a 2×2 matrix. [0][0]=[1][1]=1 (self-correlation). [0][1] or [1][0] = Pearson r between x and y.

Pandas — DataFrames & Analysis Q5–Q8
Q5Medium
Which single Pandas method returns count, mean, std, min, 25%, 50%, 75%, and max?
💡 Explanationdf.describe() returns full summary statistics for numeric columns. df.info() shows dtypes and null counts.
Q6Hard
What is the output of this code?
import pandas as pd data = {"Marks": [60, 70, 80, 90, 100]} df = pd.DataFrame(data) print(df["Marks"].var())
💡 ExplanationPandas var() uses ddof=1 (sample). Mean=80. Deviations²: 400,100,0,100,400 → sum=1000. Sample var = 1000/4 = 250.0.
Q7Medium
What does pd.cut() do in data analysis?
💡 Explanationpd.cut() discretizes a continuous variable into bins — used for grade categories (Fail/Pass/Distinction) or class intervals.
Q8Tricky
In the course code import pandas as a, what is the alias?
💡 ExplanationThe course non-standardly imported pandas as 'a'. Standard convention is 'pd'. This is a common trick question.
📊 Topic 3
Measures of Central Tendency Q9–Q12
Q9Medium
In a perfectly normal distribution, which statement is TRUE?
💡 ExplanationA symmetric normal distribution: Mean = Median = Mode, all located at the center peak.
Q10Hard
Combined mean of Group 1 (n=4, mean=60) and Group 2 (n=6, mean=80) is:
💡 ExplanationCombined mean = (n₁×mean₁ + n₂×mean₂)/(n₁+n₂) = (4×60 + 6×80)/10 = (240+480)/10 = 720/10 = 72.0.
Q11Medium
Which central tendency measure is MOST affected by extreme outliers?
💡 ExplanationThe mean includes all values in its calculation, so extreme values pull it strongly. Median and mode are resistant to outliers.
Q12Medium
The Weighted Mean formula is:
💡 ExplanationWeighted mean = Σ(weight × value) / Σ(weights). Each value is scaled by its relative importance (weight).
📏 Topic 4
Measures of Dispersion Q13–Q16
Q13Medium
Which measure of dispersion is LEAST affected by extreme values?
💡 ExplanationIQR = Q3 - Q1 uses only the middle 50% of data — completely immune to extreme outliers.
Q14Hard
Dataset A: mean=100, std=10. Dataset B: mean=50, std=8. Which has LOWER relative variability?
💡 ExplanationCV_A = 10/100×100 = 10%. CV_B = 8/50×100 = 16%. Lower CV = less relative variability. Dataset A is more consistent.
Q15Medium
Standard Deviation is the _______ of Variance:
💡 ExplanationSD = √Variance. Standard deviation is in the same units as the data; variance is in squared units.
Q16Tricky
What does ddof=1 mean in np.var(data, ddof=1)?
💡 Explanationddof=1 means "subtract 1 degree of freedom" → divide by (n-1). This gives unbiased sample variance (Bessel's correction).
〰️ Topic 5
Skewness & Kurtosis Q17–Q20
Q17Medium
A distribution with Mean > Median > Mode is:
💡 ExplanationPositive (right) skew: tail extends right. Mean is pulled toward outliers → Mean > Median > Mode.
Q18Hard
What does kurtosis(data, fisher=False) return for a normal distribution?
💡 Explanationfisher=False → Pearson kurtosis. Normal distribution has Pearson kurtosis = 3. fisher=True → excess kurtosis = 0.
Q19Medium
Leptokurtic distribution has Pearson kurtosis:
💡 ExplanationLeptokurtic: kurtosis > 3. Sharp peak, heavy tails, more outliers. Platykurtic: <3, flat peak. Mesokurtic: =3 (normal).
Q20Tricky
Karl Pearson's skewness coefficient using Mode is calculated as:
💡 ExplanationPearson's mode-based sk = (Mean − Mode) / Std. The median-based version = 3×(Mean − Median)/Std.
📈 Topic 6
Correlation & Regression Q21–Q24
Q21Medium
In simple linear regression Y = mX + c, what is model.intercept_?
💡 Explanationmodel.intercept_ = c = Y-intercept = value of Y when X=0. model.coef_[0] = slope = how Y changes per unit X.
Q22Hard
When should Spearman Rank Correlation be used over Pearson?
💡 ExplanationSpearman is non-parametric and rank-based — ideal for ordinal data, non-normal distributions, or monotonic non-linear relationships.
Q23Medium
Pearson correlation coefficient r = −0.95 means:
💡 Explanationr close to −1 = strong negative linear relationship. As X increases, Y strongly decreases.
Q24Tricky
What does pearsonr(x, y) from scipy.stats return?
💡 Explanationpearsonr() returns (r, p_value). The p-value tests statistical significance. np.corrcoef() returns only the r matrix.
🎲 Topic 7
Probability Q25–Q28
Q25Medium
A bag has 4 red, 6 blue, 10 green balls. Probability of picking a red ball = ?
💡 ExplanationP(red) = 4 / (4+6+10) = 4/20 = 0.2.
Q26Hard
P(A) = 0.4, P(B) = 0.3, P(A∩B) = 0.1. What is P(A∪B)?
💡 ExplanationP(A∪B) = P(A) + P(B) − P(A∩B) = 0.4 + 0.3 − 0.1 = 0.6. Subtract intersection to avoid double counting.
Q27Medium
For independent events A and B: P(A and B) = ?
💡 ExplanationIndependent events: P(A∩B) = P(A)×P(B). For dependent events: P(A∩B) = P(A)×P(B|A).
Q28Tricky
Empirical probability based on np.random.choice(['H','T'], size=500) is a type of:
💡 ExplanationSimulation generates observed frequencies → empirical/experimental probability. Classical uses theoretical equal-likelihood formulas.
🔔 Topic 8
Probability Distributions Q29–Q32
Q29Medium
Binomial Distribution requires all EXCEPT:
💡 ExplanationBinomial requires constant probability p across all trials. Variable probability violates the binomial assumption.
Q30Hard
In Poisson distribution, which unique property holds?
💡 ExplanationUnique to Poisson: Mean = Variance = λ (the rate parameter). This property does NOT hold for binomial.
Q31Medium
In the 68-95-99.7 rule, ~95% of data in a normal distribution falls within:
💡 Explanation68-95-99.7 rule: ±1σ = 68%, ±2σ = 95%, ±3σ = 99.7% of data in a normal distribution.
Q32Tricky
What does binom.pmf(k, n, p) calculate?
💡 Explanationpmf = Probability Mass Function = P(X = k) exactly. For cumulative P(X ≤ k), use binom.cdf(k, n, p).

Sampling & Central Limit Theorem Q33–Q36
Q33Medium
Standard Error (SE) when σ=15 and n=25 is:
💡 ExplanationSE = σ / √n = 15 / √25 = 15 / 5 = 3.0.
Q34Medium
Central Limit Theorem states that for large n, the distribution of sample means is:
💡 ExplanationCLT: For large enough n (≥30), the sampling distribution of the mean approaches Normal — regardless of the population's shape.
Q35Hard
What type of sampling does population[::10] perform?
💡 ExplanationSlicing with step [::k] selects every k-th element = Systematic Sampling.
Q36Tricky
In df.groupby('Gender').sample(2), what ensures proportional subgroup representation?
💡 Explanationgroupby + sample = stratified sampling. Gender is the strata. 2 members are independently sampled from each gender group.

In hypothesis testing, if p-value = 0.03 and α = 0.05, you should:
💡 Explanationp-value (0.03) < α (0.05) → statistically significant → Reject H₀. Note: we never "accept" H₁; we only reject or fail to reject H₀.

Blood group (O+, A+, B+, AB+) belongs to which scale of measurement?
💡 ExplanationBlood groups are unordered categories → Nominal scale. No inherent ranking between O+ and A+.

Temperature in Celsius is which scale? (0°C is NOT absence of temperature)
💡 ExplanationCelsius is Interval: equal gaps between degrees, but 0°C is arbitrary. You can't say 40°C is "twice as hot" as 20°C.

💡 ExplanationOne-sample t-test: H₀ is that the true population mean equals mu. It tests if the sample provides evidence against this claim.
